# Language Translation Tool
### CodeAlpha AI Internship — Task 1

## 📌 Project Overview
A web-based Language Translation Tool that allows users to translate text between 26+ languages using the **MyMemory Translation API**. Features a clean, professional UI with auto language detection, language swapping, and copy functionality.

## ✅ Features
- Translate between 26+ languages
- Auto-detect source language
- Swap source and target languages instantly
- Character counter (500 character limit)
- Copy translated text with one click
- Works in any browser — no installation needed
- 100% free — no API key required (MyMemory free tier)

## 🌍 Supported Languages
English, Urdu, Arabic, Hindi, Bengali, French, German, Spanish, Italian, Portuguese, Russian, Chinese, Japanese, Korean, Turkish, Dutch, Polish, Swedish, Indonesian, Malay, Thai, Vietnamese, Persian, Hebrew, Ukrainian and more.

## 🛠️ Technologies Used
| Technology | Purpose |
|-----------|---------|
| HTML5 | Structure |
| CSS3 | Styling |

## 🚀 How to Run
1. Download or clone this repository
2. Open `index.html` in any browser
3. Type text, select languages, click Translate
4. No installation or API key needed!

- Free tier: 1000 words/day
- No API key required

## 💬 How to Use
1. Enter text in the input box
2. Select source language (or use Auto Detect)
3. Select target language
4. Click **Translate**
5. Copy the result with the Copy button
